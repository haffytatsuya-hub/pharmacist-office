# 薬剤師のオフィス - Pharmacist's Office

自身が作成した画像を掲載・配布するサイトです。Apple公式ホームページ風のリキッドガラスデザインを採用しています。

## 🎨 特徴

- **Apple風デザイン**: モダンでシンプルなUI
- **リキッドガラス効果**: `backdrop-filter` を使用した洗練された背景
- **レスポンシブ対応**: スマートフォン、タブレット、デスクトップで最適化
- **3D パラレックス効果**: スクロール時に背景の医薬品アイコンが動く
- **検索機能**: 画像をタイトルで検索
- **いいねボタン**: ローカルストレージで保存
- **ダウンロード機能**: 各画像を直接ダウンロード可能

## 🚀 使い方

1. **ローカルで実行**:
   ```bash
   # ブラウザでindex.htmlを開く
   open index.html
   ```

2. **Netlify にデプロイ**:
   - GitHub にプッシュ
   - Netlify で新しいサイトを作成
   - GitHub リポジトリを接続
   - 自動デプロイが開始されます

## ✨ 機能説明

### 検索機能
画像タイトルで検索できます。リアルタイムで結果が表示されます。

### ダウンロード機能
各画像の「ダウンロード」ボタンをクリックすると、画像をダウンロードできます。

### いいねボタン
ハート型のボタンで「いいね」ができます。ローカルストレージに保存されます。

### パラレックス効果
スクロール時に背景の医薬品アイコン（💊⚕️🧪など）が異なる速度で動きます。
